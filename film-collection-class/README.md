# Film Collection Manager

Create a film collection manager class that will have the ability to:

1. Return all films in a collection
2. Show film statistics (total films, average rating, highest rated, lowest rated)
3. Show the top rated films
4. Search for films by the following filters: title, year, genre
5. Add a film to the collection

## Bonus Challenges

If you finish early, try these additional features:

1. Add validation to ensure ratings are between 0-10
2. Add a method to mark a movie as watched/unwatched
3. Add a method to update a movie's rating
4. Add a method to remove a movie from the database
5. Add a method to get movies by decade (1990s, 2000s, etc.)
6. Add a method to find duplicate movies (same title and year)
